## 1. 서론

### 1.1 목적

이 SRS는 비접촉식 AI 앰비언트 홈 케어 솔루션인 **루티드(Rooted)**의 MVP(Minimum Viable Product) 소프트웨어 시스템에 대한 기능적, 비기능적, 통신 인터페이스 및 데이터 모델 요구사항을 ISO/IEC/IEEE 29148:2018 표준 체계에 따라 확정 정의합니다.

**해결하고자 하는 문제:**

현재의 비접촉식 앰비언트 케어 시장은 B2B, B2G, B2C 구조 전반에 걸쳐 치명적인 "충족되지 못한 니즈(Unmet Need)" 상태로 정체되어 있습니다.

- **B2B (요양시설):** 저가형 모션 센서에서 양산되는 일평균 12건의 오경보로 인해 야간 간호 인력의 알람 피로도가 극대화됨. 당직 시 11건의 가짜 알람이 누적되면 경보를 끄거나 무시하게 되며, 이것이 장영희 극한 사례 발생과 같은 최악의 낙상 사망 사고 방치로 직결됨. 또한 구형 시스템은 EMR 병동 네트워크와의 연동이 단절되어 직원들의 수기 입력 이중고를 유발함. (PRD §1.1, §1.7 장영희 극한 사례)
- **B2G (지자체 공공):** 제한된 관할 예산으로 노인 돌봄 사각지대를 급히 제압해야 하나, 고사양 장비는 단가가 비싸고 기존 저사양 장비는 응급 신호를 내리 놓치는 비용-효용 딜레마 예산 장벽에 직면함. (PRD §1.5)
- **B2C (개인 보호자):** 상시 가동되는 CCTV 홈캠은 어르신의 사생활 침해 저항을 부르고, 애플워치 부류 웨어러블은 방전되면 빈껍데기가 됨. 이로 인해 타지 가족은 상시적 극도 불안감에 시달리며 독거 부모님의 골든타임 응급 이변을 알아챌 도구가 구조적으로 결여된 상태. (PRD §1.5, §1.7, §1.10)

위 문서의 대상 청자는 실무 개발팀, 통제 QA팀, 프로젝트 매니저(PM), 외부 감사관 집단입니다. 해당 명세는 개발 설계, 실제 구현, 통계 테스트, 최종 인수 승인의 유일무이한 바이블 잣대 문서로 기능합니다.

### 1.3 어휘 해설 풀이 (Definitions & Acronyms)

| 핵심 용어명 | 부연 설명 기술구 |
| :--- | :--- |
| UWB (특수 초광대역 레이더파) | 광활한 폭 대역망 통신 기술 기법체. 영상 렌즈 카메라 구속 없이 실내 반사 튕김 전파만으로 움직이는 생명체의 가슴 호흡 파동, 맥박, 신체 동선 구도로 체득 감지가 가능한 레이더 센서공학망. |
| 제로 마찰주의 (Zero-Friction 철학) | 대상자(고령자)가 본인 몸에 무언가 걸치거나 끼고, 쉴 새 없이 배터리 코드를 꼽고, 버튼을 꾹꾹 눌러대는 모든 조작적 수동 굴레에서 해방되게끔 시스템이 벽붙이 형태로 무자각 24시간 도는 절대 편의 체제 UX 억압 철폐주의 사상. |
| 헛지포 알람 (False Alarm) | 낙상이나 무호흡 등 응급 위급 사태가 단절된 상황(본인 뒤척임, 개/고양이 이동 등)을 바보 같은 기기가 위급 상태로 망상 오인하여 관리처 모바일폰에 재난경계를 울리는 악성 오동작 통보 배달. |
| 상황적 목표 추구론 (JTBD 체계망) | "사용자는 특정 상황에 닥친 고충을 뚫어줄 '과업'의 고용 도구로서 제품을 산다"는 본질주의 분석 망 프레임워크 툴킷. |
| AOS(조정 기회 포착 점수)치 | 필드 탐사 인터뷰 후 타출 집계된 보정 기회 체득 스코어링. 요구가 절실한데 시중 제품 만족도는 최악인 부분을 가중 산정하여 틈새 금맥 사이즈를 정량 지수 타산해 낸 척도표. |
| DOS(발견 기회 타격 점수)치 | 1대1 심층 VoC 발굴 기회 점수. 시중 대체 제품들보다 불만족 억눌림 분노 정도가 깊어 당사가 파고들 빈 틈새 한계 지수를 0점에서 5점까지로 타격 계량한 점수 지수판. |
| 고객 순례 지도 (CJM) | 잠재 소구자가 루티드를 인지하고 지갑을 여는 도입 단계부터 실사용 체감 이후 해지나 유지에 이르기까지 모든 생애 곡선 사이의 희로애락 여정에 깃든 감정 변화 분기를 도형화한 통계 지도. |
| 트리아지 (위급 구명 순혈 Triage 알고리즘판) | 재난 사태 등 3~4건의 응급 신호가 관제소에 동시 쇄도 폭격될 때 인공지능이 내부적으로 최상위 치명 사망 단계를 재배열 역산출하여 1순위를 최상단에 꽂아주는 목숨 우선 판별 논리 연산계. |
| 엣지 단말 지능단 (Edge AI 판독기) | 클라우드 대륙 서버까지 신호를 배달하기 전, 방안 천장의 센서 껍데기 기판 내 마그네틱 칩 안에서 스스로 작은 수준의 딥러닝 판단력을 통해 잡된 원시 지표를 걸러내는 자급자족 두뇌 처리 연산 환경 체계. |
| OTA (무형 원격 전개 펌웨어 망, Over-the-Air) | 공중 망 네트워크를 통해 기계 본체 뚜껑을 까지 않고 무선으로 최신 내부 OS 소프트웨어 판올림을 단행시켜버리는 원격 기법 조치망. |
| EMR 망 (전자의무기록/운영 서류망 시스템) | 타겟 대상인 요양원이나 실버타운 병원단에서 통용 사용하는 데스크 간호 환자 차트 관리/청구 통합 전산 인트라넷 총괄망. |
| 패킷 타전망 Webhook (웹훅선) | 우리 서버의 특정 파동 사건(이벤트 알림) 도래와 즉각 연동하여 타 기관 서버단 약속된 입구(URL 단자)로 HTTP POST 미사일 타격을 자동 호출해 알림 내용물을 떠넘겨주는 통신 접합 방식 구사망. |
| 요구사항 선별 모스코(MoSCoW) 원칙 | 스펙의 타격 강도를 지정하는 분류 방식망: Must (생명수) / Should (강력권장) / Could (옵션기구) / Won't (지금 안해) 로 나누는 우선 처리 기법. |
| 하트비트 생존 보고망 (Heartbeat) | 벽면 단말기가 정전되지 않고 살아 숨 쉬는지, 우리 관제 서버 본진단에 매 단위마다 "저 살아있소" 라 주기를 주어 통신망 타전을 쳐주는 생존 전파 확인 매커니즘. |
| 단말 판검사 수문 지능 (Validator) | 센서 날것의 AI 타산 결과치를 최종 검수하여 "이건 백 프로 개/고양이다, 이건 어르신기다"라고 거짓 헛지포 장난질을 최후로 도려내어 본국 알림을 막아내는 철문 검문소 로직 필터망. |
